การตั้งค่า system local และ keyboard configuration

บทที่ 1 การตั้งค่า System local และ keyboard configuration

การตั้งค่า locale เพื่อการกำหนดภาษาให้แก่ linux ด้วยการ แก้ไข file /etc/locale.conf หรือใช้คำสั่ง localectl

# vi /etc/locale.conf
---
LANG="en_US.UTF-8"
LC_ALL=en_US.utf-8

การใช้งาน localectl status

# localectl status
   System Locale: LANG=en_US.UTF-8
       VC Keymap: us
      X11 Layout: n/a

แสดงค่าตัวแปรของ locales variable

# localectl list-locales | grep en_
en_AG
en_AG.utf8
en_AU
en_AU.iso88591
en_AU.utf8
en_BW
en_BW.iso88591
en_BW.utf8
...

การเปลี่ยนแปลงค่า local

# localectl set-locale LANG=en_GB.utf8

# cat /etc/locale.conf
LANG=en_GB.utf8
LC_ALL=en_US.utf-8

# localectl status
   System Locale: LANG=en_GB.utf8
       VC Keymap: us
      X11 Layout: n/a

แสดงค่าตัวแปรของ list-keymaps

# localectl list-keymaps | grep ^us
us
us-acentos
us-alt-intl
us-altgr-intl
us-colemak
us-dvorak
us-dvorak-alt-intl
us-dvorak-classic
us-dvorak-intl
us-dvorak-l

การตั้งค่า keymap

# localectl set-keymap us
# localectl set-x11-keymap us
# localectl status
   System Locale: LANG=en_US.UTF-8
       VC Keymap: us
      X11 Layout: us
